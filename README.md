📚 Sistema de Gerenciamento de Cursos

Projeto desenvolvido com Java e Spring Boot para gerenciamento de cursos, permitindo cadastro, listagem, atualização e remoção de informações através de uma API REST.

🚀 Tecnologias Utilizadas
• Java 21
• Spring Boot
• Spring Web MVC
• Spring Data JPA
• Flyway
• vMariaDB
• Lombok
• Maven
• Swagger / OpenAPI

📂 Estrutura do Projeto
src/main/java/br/com/senai/gerenciamento
│
├── controller/          # Controladores da API
├── curso/               # Entidades, DTOs e repositório
├── periodo/             # Enum de períodos
└── GerenciamentoApplication.java

⚙️ Funcionalidades
✅ Cadastro de cursos
✅ Listagem de cursos
✅ Atualização de cursos
✅ Exclusão de cursos
✅ Persistência com banco de dados
✅ Versionamento de banco com Flyway
✅ Documentação automática da API com Swagger

🗄️ Banco de Dados
O projeto utiliza:
MariaDB
Flyway para controle de migrations
